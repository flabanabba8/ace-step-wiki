# Community Tools & Integrations

Curated from [awesome-ace-step](https://github.com/ace-step/awesome-ace-step).

## VST3 & DAW

- **acestep.vst3** — Official VST3 plugin (C++17/GGML/JUCE 8). CPU, CUDA, Metal, Vulkan. Build from source. ~19GB VRAM for full mode.
- **ACE-Step-DAW** — Official WIP standalone DAW with timeline, piano roll, sequencer, mixer, live-coding editor
- **gary4juce** — Third-party VST3/AU bundling six music models, lego mode for DAW audio

## Alternative UIs

- **ace-step-ui** — Spotify-inspired design, built-in audio editor, stem extraction, video generation
- **ace-step-studio** — Suno-style workflow with OpenAI-compatible LLM for lyrics + cover art
- **Tadpole Studio** — AI DJ mode, radio, library, playlists, LoRA training integration, 11 themes
- **Ace-Step-Wrangler** — DAW-inspired UI with musician-friendly sliders replacing raw parameters
- **Majik's Music Studio** — Native macOS (Swift) and Linux (GTK4) desktop app, all 7 modes, MLX acceleration

## Training Tools

- **Side-Step** — Standalone LoRA/LoKr toolkit, auto-detects model, 8GB VRAM, wizard + CLI
- **acestep-captioner** — 11B music captioning model (Qwen2.5 Omni), 1000+ instruments
- **acestep-transcriber** — Structure annotation + lyrics transcription in 50+ languages
- **Ace-Step-1.5-Dataset-Manager** — Qt/C++ desktop tool for editing training datasets

## Other Notable Projects

- **StemForge** — All-in-one: stem separation (Demucs, BS-Roformer), MIDI extraction, ACE-Step, RVC voice conversion, mixing
- **Generative Radio** — Fully local AI radio station using Qwen3 for prompts + ACE-Step for songs
- **ACE-Step-RADIO** — Continuous radio-style auto-generation stream
- **Boppy** (boppy.me) — Free hosted service, no signup, any genre up to 5 minutes
